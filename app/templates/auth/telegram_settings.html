Como configurar
  1. Pesquise @BotFather no Telegram → envie /newbot → copie o Token e cole acima → salve.
  2. Chat privado: abra o bot (t.me/nome_do_bot) e envie /start.
    Grupo: adicione o bot ao grupo e envie qualquer mensagem lá.
  3. Clique em ao lado do Chat ID para ver os chats disponíveis e selecione o desejado — ou descubra manualmente:
    https://api.telegram.org/bot<TOKEN>/getUpdates
    Abra esse link no navegador (substituindo <TOKEN> pelo seu token) e procure o campo "chat":{"id": ...}. O número (pode ser negativo para grupos) é o Chat ID.
  4. Cole o Chat ID, escolha o horário de envio e salve.
  5. Clique em Enviar teste para confirmar.
